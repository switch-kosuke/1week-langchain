## このシリーズで分かること
近年、OpenAIのChatGPTに代表される大規模言語モデル（以下、LLM）の進化は目覚ましいものがあります。しかし、LLMを単体で利用するだけでは、できることに限界があります。そこで登場するのが、LLMを活用したアプリケーション開発を支援するフレームワーク「LangChain」です。

本記事では、LangChainの概要、主要な機能、できることなどを解説します。
